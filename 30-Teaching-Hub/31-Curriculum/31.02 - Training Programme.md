# Phase 1: The Foundation (Weeks 1â€“10)
**Goal:** Master the terminal, Vim, and build the static "Artifact" (Resume).

## ðŸ“… Week 2: Linux Core (Grasshopper)

- [ ] **Mon:** Command Line, Kernel, Shell. (`echo`, `date`, `cal`)

- [ ] **Wed:** Text Manipulation. (`cat`, `less`, `grep` basics)
- [ ] **Thu:** File Operations. (`cp`, `mv`, `rm`, `mkdir`, `touch`)
- [ ] **Fri:** Symlinks (`ln -s`) and Finding files (`find`).

## ðŸ“… Week 3: Git & GitHub
**Resource:** [The Odin Project - Git Basics](https://www.theodinproject.com)
- [ ] **Mon:** Theory: Working Dir vs Staging vs Repo. Draw it.
- [ ] **Tue:** SSH Key setup. Init `learning-git` repo.
- [ ] **Wed:** The Cycle: Modify -> Stage -> Commit -> Status.
- [ ] **Thu:** Undoing things. `git checkout` & `git reset`.
- [ ] **Fri:** Remotes. Push to GitHub. Clone to new folder.

## ðŸ“… Week 4: Permissions & Processes
**Resource:** [Linux Journey - Journeyman](https://linuxjourney.com)
- [ ] **Mon:** Users & Groups. (`whoami`, `/etc/passwd`)
- [ ] **Tue:** Permissions `chmod`. Octal notation (755 vs 644).
- [ ] **Wed:** Ownership `chown`. Root privileges (`sudo`).
- [ ] **Thu:** Processes. `ps aux`, `top`. Identify PID.
- [ ] **Fri:** Signals. `kill` (15 vs 9). Background jobs (`&`).

## ðŸ“… Week 5: HTML Essentials
**Resource:** [The Odin Project - HTML](https://www.theodinproject.com)
- [ ] **Mon:** How the Web Works. DOM Inspection drill.
- [ ] **Tue:** HTML Boilerplate. Write `index.html` manually.
- [ ] **Wed:** Links (`<a>`) and Images (`<img>`).
- [ ] **Thu:** Lists (`<ul>`, `<ol>`). Build "Skills" section.
- [ ] **Fri:** **Artifact V1:** Commit raw HTML Resume to git.

## ðŸ“… Week 6: Shell Scripting I
**Resource:** [Linux Journey](https://linuxjourney.com)
- [ ] **Mon:** Shebang `#!/bin/bash` & Variables.
- [ ] **Tue:** Input/Output. `read` and `echo`.
- [ ] **Wed:** Conditional Logic (`if`, `else`).
- [ ] **Thu:** Loops (`for`, `while`). Iterate over files.
- [ ] **Fri:** **Lab:** Write `backup.sh` to zip your Obsidian vault.

## ðŸ“… Week 7: CSS Basics
**Resource:** [The Odin Project - CSS](https://www.theodinproject.com)
- [ ] **Mon:** Selectors (Class vs ID).
- [ ] **Tue:** The Box Model (Margin, Border, Padding).
- [ ] **Wed:** Flexbox Fundamentals (Container vs Item).
- [ ] **Thu:** Typography & Colors.
- [ ] **Fri:** **Lab:** Style your HTML Resume. Make it look professional.

## ðŸ“… Week 8: Networking Basics
**Resource:** [Linux Journey - Networking](https://linuxjourney.com)
- [ ] **Mon:** OSI Model (Layers 1-7 conceptual overview).
- [ ] **Tue:** IP Addresses & Subnet Masks (CIDR notation).
- [ ] **Wed:** DNS resolution. `dig google.com`.
- [ ] **Thu:** HTTP/HTTPS. Headers & Verbs.
- [ ] **Fri:** **Lab:** Use `curl -v` to debug a request to your GitHub Page.

## ðŸ“… Week 9: The "Artifact" Deployment
**Focus:** Manual Cloud Deployment
- [ ] **Mon:** GitHub Pages setup.
- [ ] **Tue:** Custom Domain theory (CNAME records).
- [ ] **Wed:** Optimization (Minify CSS concept).
- [ ] **Thu:** Accessibility Check (Lighthouse Audit).
- [ ] **Fri:** **Milestone:** Live URL for your Resume. Add to LinkedIn.

## ðŸ“… Week 10: Vim Fluency
**Resource:** [Vim Be Good](https://github.com/ThePrimeagen/vim-be-good)
- [ ] **Mon:** Relative line jumps (`10j`, `5k`).
- [ ] **Tue:** Ci (Change Inside) motions (`ci"`, `ci(`).
- [ ] **Wed:** Macros (`q`). Record a sequence and replay it.
- [ ] **Thu:** Vertical editing (Visual Block mode `<C-v>`).
- [ ] **Fri:** **Test:** Score < 400ms on basic Vim Be Good drills.
